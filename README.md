AI Study Abroad Copilot is a full-stack AI-powered platform that helps students plan their entire study abroad journey. Built with React + Next.js, TailwindCSS, and Supabase for auth and database, it features six core modules — AI University Matcher, Admission Probability Predictor, Visa Intelligence Engine, Scholarship Finder, Cost of Study Simulator, and an AI Copilot Chat powered by Claude (Anthropic) via RAG architecture.
The platform uses Supabase Auth (Google Sign-In + Email/Password), Supabase PostgreSQL for storing user profiles, applications, deadlines, and scholarship data, and Supabase Realtime for live deadline alerts. The AI layer combines Claude with a structured knowledge base of 4,200+ universities, visa policies, and scholarships to deliver personalized recommendations, animated admission probability meters, visa checklist validation, and a word-by-word streaming chat experience — all behind a premium dark-theme UI with particle animations, glassmorphism, and smooth page transitions.
